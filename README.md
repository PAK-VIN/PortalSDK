# PortalSDK

This SDK is intended to make it so an iOS/iPadOS app supports reading(and knowing what each figure is and has on it), writing and copying real data from real Skylanders and Disney Infinity figures, as well as changing the colors of the official Skylanders Trap Team Bluetooth Portal for iOS and iPadOS.

Built by Vinny "PAK-VIN" Stark with AI tools.

# Currently in Development!

## (Planned) Requirements

- iOS 18.0+ / iPadOS 18.0+
- Swift 6.0+
- Xcode 16.0+
- Bluetooth permission in `Info.plist`: `NSBluetoothAlwaysUsageDescription`
